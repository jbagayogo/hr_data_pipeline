3) Suppose one heart-rate file contains the following cleaned values: `68, 70, 71, 72, 72, 73, 74, 75, 180`. The value 180 was recorded during a sensor glitch.

a) Calculate the range of this dataset.

112

b) Explain how the extreme value affects the range.

The extreme value of 180 bpm significantly distorts the range. Without it, the range would be 75 − 68 = 7 bpm, which reflects how tightly clustered the remaining values are. The single outlier inflates the range from 7 to 112, making the data appear far more variable than it actually is. This is a key weakness of range as a statistic because it is highly sensitive to outliers since it only considers the two most extreme values in the dataset and ignores everything in between.

c) Identify a different statistic that would better represent the typical variability of the dataset. Why would this measure be better?

The interquartile range (IQR) would be a better measure of variability for this dataset. Unlike range, the IQR measures the spread of only the middle 50% of values, so the outlier of 180 bpm would not factor into the calculation at all. This makes it a more accurate representation of how the typical heart rate values actually vary.
